Researched Claude Code's cross-machine synchronization capabilities for a user managing 2 Linux notebooks and 1 Windows machine. Found that settings sync uses git-based `.claude/` directories for team-shared configs, episodic-memory plugin searches conversation history across machines when logged in, and project memory stored in CLAUDE.md files is automatically synced via git. Claude Code lacks native remote execution/teleport features; remote work requires SSH access or git-based workflows for sharing changes.
