Researched proactive tool suggestion features in Claude Code. Found that Claude does not have explicit proactive suggestion features, but uses automatic discovery mechanisms: Skills with semantic matching that activate when request descriptions match, MCP Tool Search Tool with defer_loading for dynamic discovery (85% token reduction), and @ mentions/slash commands for MCP resources and prompts. Provided configuration recommendations for optimizing Skill descriptions, using progressive disclosure, and creating tool inventory documentation to maximize discoverability.
