Fixed Chrome Canary policy configuration to block malicious sites (subdl.com, zestradar.com) from displaying popups and notifications. Used systematic debugging to identify that Chrome loads policies from multiple files that weren't all being edited; pivoted from enabling user control to site-specific blocking using PopupsBlockedForUrls, NotificationsBlockedForUrls, and JavaScriptBlockedForUrls policies. Cleaned 19 malicious cookies and 34 history entries from these sites' tracking activity. Applied Linux filesystem immutability (chattr +i) to policy files to prevent Chrome or any process from reverting the blocking rules.
