Investigated Beeper desktop app download URL by following API redirects to discover the current release (Beeper-4.2.482-x86_64.AppImage). Designed a smart update script (`update-beeper`) that checks installed vs. latest version, attempts AUR update first, then falls back to direct download and manual extraction/installation if AUR is behind. The script extracts version from the AppImage filename and handles both package manager and direct installation methods.
