Set up Superpowers plugin and Episodic Memory MCP on Windows by: (1) Installing Superpowers plugin via `claude plugin install superpowers` or `/plugin` interactive menu; (2) Adding Episodic Memory MCP to `~/.claude.json` with stdio transport pointing to the 100+ conversation archives in `claude-cross-machine-sync/episodic-memory/`; (3) Restarting Claude Code to activate. This provides searchable access to past conversations and extended AI capabilities across synced machines.
