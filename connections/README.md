# Conexões Pessoais

**Propósito**: Registrar pessoas importantes, contexto de relacionamentos e histórico de interações para manter continuidade e relevância nas conversas.

## Estrutura

Cada pessoa tem um arquivo próprio com:
- Informações básicas (nome, como se conheceram, plataforma)
- Contexto do relacionamento
- Pontos importantes mencionados
- Histórico de interações significativas
- Estilo de comunicação preferido

## Como Usar

Quando ajudando com comunicação pessoal:
1. Consultar o arquivo da pessoa para contexto
2. Manter o tom e estilo apropriados
3. Referenciar conversas anteriores quando relevante
4. Atualizar o arquivo após interações significativas

## Princípios

- **Privacidade**: Informações sensíveis ficam locais, nunca em repositórios públicos
- **Contexto**: Entender a dinâmica antes de sugerir respostas
- **Autenticidade**: Ajudar a articular pensamentos, não criar personas falsas
- **Continuidade**: Manter registro para não repetir perguntas ou esquecer detalhes
