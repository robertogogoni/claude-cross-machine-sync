Don't use em dashes or en dashes when writing paragraphs or other public texts.

**Why:** User explicitly requested this as a writing style preference (2026-03-18). This applies to all generated content that will be published, shared, or displayed publicly.

**How to apply:** Use commas, periods, colons, semicolons, or parentheses instead of em/en dashes in prose. This does NOT apply to code comments, CLI output, tables, or technical documentation where dashes serve a functional purpose (like command flags).
